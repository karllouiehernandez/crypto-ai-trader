Risk Management

ATR-based position sizing. Two independent circuit breakers prevent catastrophic loss.

ATR Sizing:
  qty = (equity × 1%) / (ATR × 1.5)

Circuit Breakers:
  Daily Loss   → halt if loss > 3%
  Drawdown CB  → halt if equity < peak × 85%

Acceptance Gates (walk-forward):
  Sharpe    ≥ 1.5
  Max DD    ≤ 20%
  Prof.Fac  ≥ 1.5
  Trades    ≥ 200

LLM Confidence Score — How It's Derived

Claude evaluates the strategy holistically from a quantitative + qualitative lens:

  • Walk-forward pass rate (% of windows passing acceptance gate)
  • Sharpe stability across rolling windows (low variance = higher confidence)
  • Regime coverage (does it degrade in certain regimes?)
  • Known weaknesses from KB (strategy_learnings.md context)
  • Parameter sensitivity (did ±20% changes shift results?)

A strategy can pass the quantitative gates but receive a low LLM confidence if Claude detects regime-specific brittleness or overfitting signals in the KB history.

LLM Recommendation States

ABANDON
Sharpe < 1.0 OR confidence < 0.4 — strategy fundamentally broken
↑ escalation path
HOLD_PAPER
Some gates failing — continue paper trading, apply param suggestions
↑ escalation path
PROMOTE_TO_LIVE
All 5 gates pass — 3 consecutive evaluations needed before dashboard badge

Auto-Promote Safety

LLM_AUTO_PROMOTE=false by default. The system recommends, the user decides. Set LLM_AUTO_PROMOTE=true in .env to allow autonomous promotion — use only after you trust the system's 30+ day track record.

Multi-Agent Architecture (Sprint 12)

Each symbol gets its own independent TokenAgent asyncio task. Agents never execute trades — they only report signals. The AgentCoordinator aggregates reports via an asyncio.Queue and provides the latest signal per symbol to PaperTrader via a non-blocking dict lookup. This decouples signal generation from trade execution.

TokenAgent (per symbol)

class TokenAgent:
  symbol: str
  strategy: StrategyBase | None
  result_queue: asyncio.Queue

  async def run():
    while True:
      report = await _step()
      await queue.put(report)
      await asyncio.sleep(1.0)

for sym in SYMBOLS:

Strategy Assignment to Agents

Mode 1: Heuristic (Default)
All agents use compute_signal().
No plugin strategy assigned. Full fallback to existing regime-routing logic.
Mode 2: Named Plugin
--strategy rsi_v1 — all agents use the same StrategyBase subclass loaded from strategies/rsi_v1.py.
Mode 3: LLM-Generated
llm/generator.py creates strategies/generated_{ts}.py → watchdog hot-loads → coordinator assigns to agents automatically.
